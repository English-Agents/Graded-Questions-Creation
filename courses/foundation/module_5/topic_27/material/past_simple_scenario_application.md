#Past Simple: Application Scenario

##Information about Completed Past Action

The past simple tense is used to describe **actions or events that were finished at a particular time in the past**. It indicates that something occurred previously and is no longer ongoing.

|               | Noun                                      | Sentence Structure                          | Examples                                                 |
|---------------|------------------------------------------|---------------------------------------------|----------------------------------------------------------|
| Positive      | I/ you/ we/ any plural noun/singular noun/they/that/this/those/these/he/she/it | + verb (II form)                           | She traveled to France last summer.                      |
|               |                                          |                                             | They completed the project on time.                      |
|               |                                          |                                             | He cooked dinner for the whole family.                   |
| Negative      | I/ you/ we/ any plural noun/singular noun/they/that/this/those/these/he/she/it | did not + verb (I form)                    | She did not travel to France last summer.                |
|               |                                          |                                             | They did not complete the project on time.               |
|               |                                          |                                             | He did not cook dinner for the whole family.             |


##Past Habits 

The simple past tense is used to describe actions, habits, or situations that **were true or occurred regularly in the past but do not happen or are no longer true now**. It helps convey things that were once part of someone's routine or a previous state that has since changed.

|               | Noun                                     | Sentence Structure                                    | Examples                                                  |
|---------------|------------------------------------------|------------------------------------------------------|----------------------------------------------------------|
| Positive      | I/ you/ we/ any plural noun/singular noun/they/that/this/those/these/he/she/it | + used to + verb (I form) + rest of the sentence     | She used to play the piano every evening.                |
|               |                                          |                                                      | They used to visit their grandparents every summer.      |
|               |                                          |                                                      | He used to ride his bike to school.                      |
| Negative      | I/ you/ we/ any plural noun/singular noun/they/that/this/those/these/he/she/it | + did not use to + verb (I form) + rest of the sentence | She did not use to play the piano when she was younger.   |
|               |                                          |                                                      | They did not use to visit their grandparents during the holidays. |
|               |                                          |                                                      | He did not use to ride his bike to school before the new bike lanes were built. |


##Information about the Status of a Past Activity

We use simple past tense sentences to highlight **the status or result of a past activity**. These sentences tell us what happened to someone or something and provide an update on the condition or outcome of the action in the past.

|               | Noun                                      | Sentence Structure                            | Examples                                                |
|---------------|------------------------------------------|-----------------------------------------------|---------------------------------------------------------|
| Positive      | I/ He/ She/ That/ This/ any singular noun (except you) | + was + verb (III form) + rest of the sentence.   | The book was read by many students last year.           |
|               | You/ They/ We/ Those/ These/ any plural noun | + were + verb (III form) + rest of the sentence.   | The invitations were sent to all the guests last week.  |
| Negative      | I/ He/ She/ That/ This/ any singular noun (except you) | + was not + verb (III form) + rest of the sentence. | The letter was not delivered to the correct address.    |
|               | You/ They/ We/ Those/ These/ any plural noun | + were not + verb (III form) + rest of the sentence. | The cookies were not baked in time for the party.       |

##Past Identity or Status

The simple past tense can describe a **previous identity, status, or condition that existed in the past** but is not true anymore. When referring to someone's former identity or how something used to be, we often use **"was" or "were"** to convey this.

|               | Noun                                      | Sentence Structure                     | Examples                                             |
|---------------|------------------------------------------|----------------------------------------|------------------------------------------------------|
| Positive      | I/ He/ She/ That/ This/ any singular noun (except you) | + was + information.                  | She was a teacher before changing careers.           |
|               | You/ They/ We/ Those/ These/ any plural noun | + were + information.               | They were friends for many years.                    |
| Negative      | I/ He/ She/ That/ This/ any singular noun (except you) | + was not + information.              | She was not a teacher during that time.              |
|               | You/ They/ We/ Those/ These/ any plural noun | + were not + information.          | They were not friends when they first met.           |
