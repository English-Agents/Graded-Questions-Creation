# Past Perfect and Past Perfect Continuous Application

#Past Perfect and Continuous Tense : Scenario Application

##Past Perfect Tense

###Using “Had”

| **Structure Type** | **Subject** | **Structure** | **Examples** |
|--------------------------|------------------|------------------------------------|-----------------------------------------------------|
| **Positive** | I/you/ we/ any plural noun/ those/ these/he/ she/ it/ any singular noun/this/ that | Subject + had + V3 | She had completed her homework before dinner. 
| | | |We had finished cleaning the house before guests came.|
| | | |I had seen the movie before we talked about it. |
| **Negative** | I/you/ we/ any plural noun/ those/ these/he/ she/ it/ any singular noun/this/ that | Subject + had not + V3 | She had not completed her homework before dinner. 
| | | |We had not finished cleaning the house before guests came. |
| | | |I had not seen the movie before we talked about it. |

###Using “Had Been”

| **Structure Type** | **Subject** | **Structure** | **Examples** |
|--------------------------|------------------|------------------------------------|-----------------------------------------------------|
| **Positive** | I/you/ we/ any plural noun/ those/ these/he/ she/ it/ any singular noun/this/ that | Subject + had been + V3 | She had been invited to the party before it started. |
| | | | They had been given clear instructions before the test began. |
| | | | I had been informed about the changes before the meeting. |
| **Negative** | I/you/ we/ any plural noun/ those/ these/he/ she/ it/ any singular noun/this/ that | Subject + had not been + V3 | She had not been invited to the party before it started. |
| | | | They had not been given clear instructions before the test began. |
| | | | I had not been informed about the changes before the meeting. |

##Past Perfect Continuous Tense

| **Structure Type** | **Subject** | **Structure** | **Examples** |
|--------------------------|------------------|------------------------------------|-----------------------------------------------------|
| **Positive** | I/you/ we/ any plural noun/ those/ these/he/ she/ it/ any singular noun/this/ that | Subject + had been + verb-ing | She had been studying for hours before the test began. |
| | | | They had been playing football before it started to rain. |
| | | | I had been waiting for the bus for an hour before it arrived. |
| | | | You had been working hard before you finally took a break. |
| **Negative** | I/you/ we/ any plural noun/ those/ these/he/ she/ it/ any singular noun/this/ that | Subject + had not been + verb-ing | She had not been studying for hours before the test began. |
| | | | They had not been playing football before it started to rain. |
| | | | I had not been waiting for the bus for an hour before it arrived. |
| | | | You had not been working hard before you finally took a break. |

**Conversation Examples**

Person A: The garden looks so lush and green! Had it been raining all week?
Person B: ""Yes, it **had been raining** for days before the sun finally came out, but the lawn **had not been mowed** in a while, so it looks overgrown.

Person A: ""Had you been practicing for the performance before it started?""
Person B: ""Yes, I **had been practicing** all morning, but I **had not memorized** the last part perfectly.""
 
Person A: ""Why was Maria in such a rush this morning? Had she been running late?""
Person B: ""Yes, she **had been rushing** to get ready, but she **had not packed** her bag the night before, so it took her longer to leave.""
